Sales Analysis BI Project

Анализ розничных продаж для выявления ключевых драйверов прибыли и убыточных зон.


Бизнес-вопросы

1. Где компания зарабатывает больше всего?
2. Какие продукты приносят убыток — и почему?
3. Есть ли регионы с высоким объёмом продаж, но низкой прибылью?


 Технологии

- SQL (SQLite) — для агрегации и фильтрации данных.
- Tableau — для визуализации и сторителлинга.



Ключевые инсайты

Technology — лидер по прибыли и продажам.  
Убыточные продукты — многие из категории Technology. Возможно, проблема в логистике или ценообразовании.  
Регионы — Западная Европа и Северная Америка — самые прибыльные.
